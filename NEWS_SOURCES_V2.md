## 🤖 自动化搜索策略

### 策略设计原则

**1. 分层搜索**：按Tier优先级依次搜索
**2. 关键词组合**：公司名 + 业务线 + 时间过滤
**3. 去重合并**：多源信息交叉验证

#### Tier 3：行业媒体（主力自动化）

**搜索关键词设计**：

```python
# 核心竞对（5个）
competitors = {
    "字节跳动": ["字节跳动", "抖音", "巨量引擎", "Seedance"],
    "小红书": ["小红书", "蒲公英", "种草"],
    "快手": ["快手", "磁力引擎", "磁力金牛"],
    "腾讯": ["腾讯", "混元", "视频号广告"],
    "阿里": ["阿里", "阿里妈妈", "淘宝", "天猫"]
}

# 行业赛道（3个）
industries = {
    "游戏": ["游戏广告", "IAA", "买量"],
    "本地生活": ["本地生活", "到店", "美团", "抖音生活服务"],
    "电商": ["电商广告", "直播带货", "货架电商", "GMV"]
}

# 媒体站点
media_sites = [
    "36kr.com",
    "finance.sina.com.cn",
    "tmtpost.com",
    "huxiu.com",
    "adquan.com",
    "newrank.cn",
    "pai.com.cn"
]

# 组合搜索
for company, keywords in competitors.items():
    for site in media_sites:
        query = f"site:{site} ({' OR '.join(keywords)}) 商业化 OR 广告 OR 收入 after:2026-03-11"
        search(query)
```

## 🔄 每日更新工作流

### 时间：每天早上 8:00

```
Step 1: Tier 2 官方公众号搜索（20分钟）
  ↓ 搜索11个公众号的最新文章
  ↓ LLM筛选3-5条最重要的

Step 2: Tier 3 行业媒体搜索（15分钟）
  ↓ 搜索7个媒体网站
  ↓ LLM提取12条核心新闻（每公司3条）

Step 3: Tier 1 官方资料检查（每周1次，5分钟）
  ↓ 访问官方学习平台
  ↓ 查看有无重大产品更新

Step 4: 交叉验证与洞察生成（10分钟）
  ↓ LLM分析最新动态
  ↓ 生成3条战略洞察

Step 5: 更新HTML并提交（5分钟）
  ↓ 更新index.html
  ↓ 提交到GitHub
  ↓ 自动部署

总耗时：约50分钟/天（自动化后缩短到5分钟）
```
