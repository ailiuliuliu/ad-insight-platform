# 🚀 快速上传指南（浏览器操作）

## 📦 需要上传的文件

以下文件已经在 `ad-insight-demo/` 目录下准备好了：
- ✅ `index.html` - 主页面
- ✅ `favicon.svg` - 网站图标
- ✅ `README.md` - 项目说明

## 🌐 浏览器上传步骤

### 第 1 步：登录并进入仓库

1. 访问：https://github.com/ailiuliuliu/ad-insight-platform
2. 如果未登录，点击右上角 **Sign in** 登录
3. 登录后，页面会显示 "This repository is empty."

### 第 2 步：上传文件

在空仓库页面，你会看到：

```
Quick setup — if you've done this kind of thing before

...or create a new repository on the command line
...or push an existing repository from the command line
...or import code from another repository

Get started by creating a new file or uploading an existing file.
```

点击 **"uploading an existing file"** 链接

### 第 3 步：选择文件

1. 进入上传页面后，拖拽或点击选择文件
2. 选择以下 3 个文件（按住 Cmd 多选）：
   - `index.html`
   - `favicon.svg`
   - `README.md`
   
   文件位置：`~/Documents/超级小李/ad-insight-demo/`

3. 等待文件上传完成

### 第 4 步：提交

1. 在页面底部 "Commit changes" 区域：
   - Commit message: `初始化商业化洞察平台`
2. 点击 **"Commit changes"** 按钮

### 第 5 步：启用 GitHub Pages

1. 点击仓库顶部的 **Settings** 标签
2. 在左侧菜单找到 **Pages**
3. 在 "Source" 部分：
   - Branch: 选择 `main`
   - Folder: 选择 `/ (root)`
4. 点击 **Save**
5. 等待约 30-60 秒

### 第 6 步：访问你的网站

访问：**https://ailiuliuliu.github.io/ad-insight-platform/**

🎉 完成！
